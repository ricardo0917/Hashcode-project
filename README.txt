para correr el programa sin IDE, busque las siguientes direcciones en Program Files, las copia y pega entre comillas
"C:\Program Files (x86)\Java\jdk-14.0.1\bin\javac"
"C:\Program Files (x86)\Java\jdk-14.0.1\bin\java"
en el archivo compilar.bat que le dejare ahi profesor, y al lado de javac escribe el nombre de la clase a ejecutar o sea HashCode.java
al lado de java, escribe solo HashCode, guarda como compilar.bat y lo corre si no da ningun error da Enter y debe correr el programa.

quedaria asi al final: "C:\Program Files (x86)\Java\jdk-14.0.1\bin\javac" HashCode.java

                       "C:\Program Files (x86)\Java\jdk-14.0.1\bin\java" Hashcode
